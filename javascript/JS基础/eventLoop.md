4. 进程和线程区别

   ```
   一个进程可以有多个线程，比如渲染线程，JS引擎线程，等。进程表示一个程序，线程是进程中的单位
   
   多线程在单核CPU中其实也是顺序执行的，不过系统可以帮助你切换那个执行而已，没有提高速度
   
   单线程的优点: 解决切换上下文的时间，锁的文件，节省内存。
   
   
   node优点可以开启多个子进程，里面包含着一个线程
   ```
